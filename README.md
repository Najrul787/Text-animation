## QTextAnimation

#### Use
```javascript $(selector).animQText(options); ```

#### code
```javascript
var options = {
   effectIn: 'rollIn',
   effectOut: 'rollOut',
   delay:  800,
   duration: 500,
   revers: false,
}
$('.demo').animQText(options);
```
You can use [Animate.css](https://daneden.github.io/animate.css/) all classes for `` effectIn  `` & `` effectOut `` Animation
